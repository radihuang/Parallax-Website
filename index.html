視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景比前景移動得更慢。這在訪客向下捲動頁面時產生類似 3D 的效果，增加深度感並打造更沉浸的瀏覽體驗。視差源自視覺錯覺。因為人眼會把靠近的物體看得 比遠處的物體大，我們也會覺得遠處的物體移動得比較慢。這種錯覺長久以來被應用在各種媒介的視 差技術中，營造逼真的感受。最早可以追溯到傳統動畫，例如迪士尼的《白雪公主和七矮人》，以及 《超級瑪利歐》等電玩遊戲。隨著 CSS 與 HTML 的進步，視差效果最終發展成為今日常見的網頁設計 元素。

最早的瀏覽器視差滾動實作之一，由網頁開發者 Glutnix 在 2007 年的部落格文章中分享，內含 JavaScript 與 CSS 2 的範例程式碼與示範，甚至支援當時的 Internet Explorer 6 及其他瀏覽器。 不過直到 2011 年 HTML5 與 CSS3 出現後，這項技術才在網頁設計師之間廣受歡迎。支持者認為，這是 擁抱網路流動性最簡單的方式之一。他們把視差背景當成提升互動、改善網站整體體驗的工具。然 而，普渡大學在 2013 年發表的研究指出：「……雖然視差滾動增強了使用者體驗的某些面向，但未必 能改善整體的使用者體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像有兩個主要理由：

  • 有助於說故事

    視差圖像非常適合說故事。良好的設計重在溝通——當我們打造網站時，希望與訪客互動，講述關於 產品、服務或品牌的動人故事。視差滾動能讓訪客沉浸在獨一無二的線上體驗中。甚至可以只用一頁 網站，就讓使用者不用切換頁面也能讀完完整的品牌故事。

  • 有助於提升使用者參與

    視差圖像也能提升使用者參與度。降低網站的跳出率是網頁設計時必須留意的目標之一。跳出率是指造訪 者只瀏覽單一頁面後就按下「返回」或關閉分頁的比例。過高的跳出率會傷害搜尋排名，因此任何能 降低跳出率的作法都值得嘗試。精心設計的視差頁面提供動態體驗，能讓使用者願意停留更久。


想在網站中加入視差滾動效果時，可以參考以下建議。

  • 精準地使用

    與其在所有頁面或任何時候都使用視差滾動，不如把它運用在特定元素上效果更好。可以放在頁首、標題， 或僅限首頁。你希望視差滾動強化網站設計，而不是讓使用者分心。

  • 壓縮圖片

    視差滾動會結合各式媒體檔案與 CSS 形狀來營造深度感。別讓過大的圖片檔拖慢載入速度，進而影響網站 效能與使用者體驗。

  • 玩轉色彩

    視差滾動不僅與前景、背景圖像有關，色彩同樣關鍵。背景可以使用較低飽和的色調，前景則運用鮮明的顏色， 以營造更強烈的層次感。
